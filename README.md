# 📝 Todo App — React + Tailwind CSS

A simple and clean Todo App built using React.  
This app helps users manage daily tasks efficiently with a responsive design and local storage support.

---

## 🚀 Features

- Add, edit, and delete tasks easily  
- Mark tasks as completed  
- Data is stored in browser using Local Storage  
- Fully responsive (works on mobile, tablet, desktop)  
- Clean UI using Tailwind CSS  
- Fast and smooth performance with React  

---

## 🛠️ Tech Stack

- React (Functional Components + Hooks)  
- Tailwind CSS  
- JavaScript (ES6+)  
- Local Storage (for saving tasks)  

---

## 📱 Responsiveness

The app is designed to work on all screen sizes:
- Mobile phones  
- Tablets  
- Laptops and desktops  

---

## 💾 Local Storage

All tasks are saved in the browser using Local Storage.  
This means:
- Tasks remain even after refreshing the page  
- No backend or database is required  

---

## 📌 Features Overview

- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed/incomplete  

---

## ▶️ How to Run the Project

1. Clone the repository  
```bash
git clone https://github.com/adarshj61/todo-app.git
```

2. Navigate to the project folder  
```bash
cd todo-app
```

3. Install dependencies  
```bash
npm install
```

4. Start the development server  
```bash
npm run dev
```
